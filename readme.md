## 特征

-   语法高亮显示开箱即用. 
-   快速静态搜索所有文档. 
-   多个`git repos`交叉编译. 
-   每页的静态JSON数据. 
-   自动生成的目录. 
-   SEO友好 - 一切都是静态的!

> 噢, 有些特征没看懂, 来 [这里吧](https://gitdocs.netlify.com/)
